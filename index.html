What Exactly is Free Code Camp?

  • An open source community where you learn to code and practice by building projects for nonprofits
  • And a lot more.
  • We believe technology education should be effective, open source, and free.

How does Free Code Camp work?

  • We work through a full stack JavaScript curriculum together.
  • We practice by building projects for nonprofits together.

Questions?

  • Also, we are doing a huge survey of people who are new to code.
